# Desk.fm
## Simple ways to play your music *at your desk*

DeskFM was a little thing I started earlier this week since my Apple Music was about to expire and I had some legal music I wanted to play.
Anyways I didn't like the built in Windows thing or VLC so I decided to make my own little player.

Essentially you've got 2 view modes to show songs, you can also make playlists and queue songs. There's also crossfade but I was never
able to get it to work, so I've quite literally just abandoned that feature right there.

I really think this was a cool project to make since I needed my thing pretty quickly and my python skills are pretty good, so this worked
out in my favour but yeah. Lmk what you think.

## Installing

For now, I haven't made a compiled version, but simply just run
```python
pip install -r requirements.txt
```
and that should install all the requirements. Then you can just run
```python
python main.py
```
and then DeskFM should open. If it dosen't then please make an issue and I'll fix it.
